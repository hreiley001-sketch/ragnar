# Event Bus as Nervous System

Events are the shared signal layer — persisted for the Command Hub, mirrored for analytics.

## Why it matters

Without a nervous system, features become isolated organs. The bus lets rides, payments, and attention share one language of “what just happened.”

## Vault parallel

Links are the vault’s event bus. Meaningful `[[wikilinks]]` propagate awareness across notes the way events propagate across BirdmanOS.
