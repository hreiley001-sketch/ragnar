# Maps of Content

A map is an orientation note — not a folder substitute.

## What a map does

- Shows relationships between ideas
- Reveals missing links
- Gives a place for orphans to land

## What a map is not

- A table of contents that never changes
- A dumping ground for every file in a folder

## In this vault

Start from [[Maps/Atlas]]. Domain maps ([[Maps/BirdmanOS]], [[Maps/RAGNAR]], [[Maps/Creative Flow]]) specialize without fragmenting the mind.
