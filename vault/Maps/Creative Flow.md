# Creative Flow

How ideas move in this vault — and in the mind that uses it.

## The glide

```
Capture → Clarify → Connect → Compost → Create
```

| Step | Feels like | Lives in |
|---|---|---|
| Capture | Instant, judgment-free | [[Inbox]] · [[Daily]] |
| Clarify | One idea, one note | [[Templates/Evergreen]] |
| Connect | Links in multiple directions | [[System/Linking Conventions]] |
| Compost | Review, prune, merge | [[System/Weekly Review]] |
| Create | Projects with outcomes | [[Projects/Index]] |

## Instincts this system honors

- Clarity over volume
- Pattern recognition over filing
- Conceptual linking over folders
- Cohesion over completeness
