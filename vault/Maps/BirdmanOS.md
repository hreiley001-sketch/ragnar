# BirdmanOS

The product’s operating mind: rides, events, control surfaces — structured experience without rigidity.

## Essence

BirdmanOS turns marketplace moments into **rides** — timed phases that feel like flow, not forms.

```
idle → lobby → showcase → bidding → cooldown → archived
```
