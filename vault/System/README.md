# README

# Birdman Flow Vault

Minimal Obsidian ecosystem for cohesive thinking — personal creative flow and RAGNAR / BirdmanOS knowledge in one mind.

## Open in Obsidian

1. Obsidian → **Open folder as vault**
2. Choose `vault/` (this folder)
3. Start at [[Home]]

## Layout (six homes)

```
vault/
  Home.md
  Inbox/          capture
  Daily/          temporal flow
  Evergreen/      atomic ideas
  Projects/       active vessels
  Maps/           orientation
  Templates/      frictionless starts
  System/         principles + skill path
```

## First hour

1. Read [[System/Flow Principles]]
2. Open today’s daily (or create from [[Templates/Daily]])
3. Drop three thoughts in [[Inbox]]
4. Process one into evergreen
5. Glance at [[Maps/Atlas]]

## Plugins (suggested, optional)

Core is enough to start. Later: Templates (core), Daily notes (core), Graph view (core). Community only when friction appears.

## Energy

No branding layer. Clean structure. Obsessive interconnection. Flow over form.
