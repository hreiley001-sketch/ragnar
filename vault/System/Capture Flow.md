# Capture Flow

From spark to placed idea — a glide, not a grind.

## Fast path (during the day)

1. Open [[Daily]] note or [[Inbox]]
2. Write the thought in one breath
3. Optional: one `[[link]]` if it jumps out
4. Move on

## Process path (when attention is free)

1. Read inbox / daily captures
2. Decide: fleeting · evergreen · project · map · discard
3. If evergreen/project/map → create from template, move substance, link neighbors
4. Clear the capture line so Inbox stays empty-ish

## Promotion tests

| Signal | Destination |
|---|---|
| Reusable idea | [[Evergreen/Index]] |
| Has an outcome | [[Projects/Index]] |
| Cluster of related notes | [[Templates/Map]] |
| Still raw | Stay in Inbox |
