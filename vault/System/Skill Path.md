# Skill Path

Progressive Obsidian mastery. Do the level you’re on. Don’t skip the feel.

## Level 1 — Capture without fear

**Practice:** 7 daily notes. Use [[Templates/Daily]]. Empty [[Inbox]] twice.

**Win condition:** You trust the vault with raw thought.

## Level 2 — Atomic linking

**Practice:** Create 5 evergreen notes from [[Templates/Evergreen]]. Each has ≥2 links.

**Win condition:** Graph View shows a small constellation, not islands.

## Level 3 — Maps as orientation

**Practice:** Maintain [[Maps/Atlas]] + one domain map. Add a new map only when a cluster asks for it.

**Win condition:** You navigate by maps, not by folder digging.

## Level 4 — Projects as vessels

**Practice:** Run one project from start → harvest ([[Templates/Project]]).

**Win condition:** Closing a project leaves evergreen residue, not a dead folder.

## Level 5 — System fluency

**Practice:** Run [[System/Weekly Review]] for 4 weeks. Adjust only what reduces friction.

**Win condition:** The vault feels like thinking — not administering notes.

## Exercises

| Exercise | How |
|---|---|
| Orphan hunt | Graph View → find unlinked → give each one home link |
| Split drill | Take a fat note → extract one atomic idea |
| Bidirectional pass | Open a note → ensure neighbors acknowledge it |
| Ride metaphor | Map a workflow into phases like [[Evergreen/Ride as Flow State]] |
