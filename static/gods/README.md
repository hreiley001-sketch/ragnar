# Realm riches — Asgard treasure accents

Each realm carries a **patron color palette**. Color pops come from **floating gems**
in the Asgard atmosphere — not from gate cards or statues.

| Realm key   | Patron   | Halls            | Accent · gems |
|-------------|----------|------------------|---------------|
| `freyr`     | Freyr    | Marketplace      | Amber-gold · jade |
| `thor`      | Thor     | Live Rooms       | Ice-blue · gold bolt |
| `tyr`       | Týr      | Groups           | Jade · oath-gold |
| `loki`      | Loki     | The Feed         | Violet · ember |
| `heimdall`  | Heimdall | Stores & Halls   | Teal · bifrost |
| `odin`      | Odin     | Command Hub / default | Steel-gold · steel |

Homepage: pantheon wash + **found gem caches** (one tight pile per patron palette) nested in the vault.
Realm pages: 3–4 patron caches + soft bloom.

Gems sit in clustered piles with a shared shimmer — not a random float field.
