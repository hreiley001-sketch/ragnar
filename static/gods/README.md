# God art — drop your illustrations here

Each realm's patron god is shown as a **stylized glowing statue** in the background
of its pages (with that god's runic glow, jewels, and color accents woven through
the page). All six appear on the homepage "Pantheon" wall.

Until a PNG exists, a carved stylized silhouette (unique helm, weapon, companion,
and jewel accents) stands in.

## To add real art
Export each god as a **transparent-background PNG**, tall/vertical, and save it here
with the exact filename below. It will appear automatically — no code changes.

| File to add            | God      | Realm            | Accent · jewels |
|------------------------|----------|------------------|-----------------|
| `freyr.png`            | Freyr    | Marketplace      | Amber-gold `#c8901f` · jade |
| `thor.png`             | Thor     | Live Rooms       | Ice-blue `#2f93c4` · gold bolt |
| `tyr.png`              | Týr      | Groups           | Jade `#2f8f6b` · oath-gold |
| `loki.png`             | Loki     | The Feed         | Violet `#7a5cc4` · ember |
| `heimdall.png`         | Heimdall | Stores & Halls   | Teal `#1f8f9a` · bifrost |
| `odin.png`             | Odin     | Command Hub / default | Steel-gold `#a67c1a` · steel |

## Tips
- Transparent background (PNG) so the altar glow and jewels show around the figure.
- Roughly 800×1200px or larger; vertical framing (full figure).
- The altar already supplies glow, light-rays, corner jewels and colour — plain,
  well-lit figures on transparent backgrounds work best.
- Filenames are lowercase, exactly as above.
