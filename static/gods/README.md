# God art — drop your illustrations here

Each realm's patron god is shown as a **glowing statue** in the background of its
pages, and all six appear on the homepage "Pantheon" wall.

## To add real art
Export each god as a **transparent-background PNG**, tall/vertical, and save it here
with the exact filename below. It will appear automatically — no code changes.
Until a file exists, a carved silhouette placeholder stands in its place.

| File to add            | God      | Realm            | Accent colour |
|------------------------|----------|------------------|---------------|
| `freyr.png`            | Freyr    | Marketplace      | Amber-gold `#c8901f` |
| `thor.png`             | Thor     | Live Rooms       | Ice-blue `#2f93c4` |
| `tyr.png`              | Týr      | Groups           | Jade `#2f8f6b` |
| `loki.png`             | Loki     | The Feed         | Violet `#7a5cc4` |
| `heimdall.png`         | Heimdall | Stores & Halls   | Teal `#1f8f9a` |
| `odin.png`             | Odin     | Command Hub / default | Steel-gold `#a67c1a` |

## Tips
- Transparent background (PNG) so the altar glow shows around the figure.
- Roughly 800×1200px or larger; vertical framing (full figure).
- The altar already supplies the glow, light-rays and colour — plain, well-lit
  figures on transparent backgrounds work best.
- Filenames are lowercase, exactly as above.
